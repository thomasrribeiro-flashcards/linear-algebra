# linear-algebra

Flashcards for Linear Algebra — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

## Coverage

| File | Topic |
|------|-------|
| 01_linear_systems | Linear equations, matrices, row operations, echelon form, RREF, Gaussian elimination |
| 02_matrix_algebra | Matrix operations, multiplication, transpose, inverse, elementary matrices |
| 03_determinants | Cofactor expansion, row-reduction, properties, Cramer's rule, adjugate |
| 04_vector_spaces | Vector space axioms, subspaces, span, null/column/row space |
| 05_basis_dimension | Linear independence, basis, dimension, rank, rank-nullity theorem |
| 06_coordinates_change_of_basis | Coordinate vectors, change-of-basis matrix, isomorphism |
| 07_linear_transformations | Kernel, range, standard matrix, composition, similarity |
| 08_eigenvalues_diagonalization | Characteristic polynomial, eigenspaces, diagonalization $A = PDP^{-1}$ |
| 09_inner_products_orthogonality | Dot product, norm, orthogonal sets/bases, orthogonal complement, projection |
| 10_gram_schmidt_qr | Gram-Schmidt process, QR decomposition |
| 11_least_squares | Normal equations, least-squares solution, regression |
| 12_symmetric_spectral_quadratic | Symmetric matrices, spectral theorem $A = PDP^T$, quadratic forms, definiteness |
| 13_svd | Singular value decomposition, pseudoinverse, low-rank approximation |
